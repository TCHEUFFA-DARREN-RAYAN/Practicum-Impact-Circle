# Roadmap

## Completed ✅

- Volunteer registration, profile, and verification
- Organization registration and gig posting
- Admin verification queue with approve/reject
- Gig browsing with category and search filters
- Time-of-day and hours filters on gig search
- Application flow with personal statement
- Apply button "Waiting for Approval" state after submitting
- Task workflow: Start Task → Claim Hours → org approval
- Recurring gigs (daily / weekly / monthly) with time-of-day settings
- Org applicant detail page with full volunteer profile
- Admin user management: view, block/unblock, soft-delete
- Admin organization management: view, edit, soft-delete
- Org volunteer tracker with session counts and hours
- Star rating system for volunteers after task approval
- Points and rewards system
- Soft-delete (archive) for gigs
- `isBlocked` enforcement on every authenticated request
- Email notifications for verification, application decisions, and task approvals
- TLS-flexible email service for corporate proxy environments

## Planned 🔜

- **Messaging system** — in-app direct messages between org and volunteer
- **Calendar integration** — iCal export for upcoming shifts
- **Volunteer leaderboard** — top volunteers by category, hours, and points
- **CSR dashboard** — corporate partner view of employee volunteer stats
- **Push notifications** — browser push for real-time updates
- **Multi-language support** — English and French (bilingual for New Brunswick)
- **Document expiry alerts** — remind volunteers when gov ID or background check expires
- **Waitlist** — allow volunteers to join a waitlist when gig spots are full
- **QR check-in** — org scans volunteer QR code to mark attendance at the event
- **Impact reports** — exportable PDF summary of org volunteer activity
